# FitBuddy AI Fitness Plan Generator

FitBuddy is a web application that generates personalized workout plans based on age, weight, and fitness goals.

## Technologies Used
- Python
- FastAPI
- HTML/CSS
- Uvicorn

## Features
- Enter Age
- Enter Weight
- Select Fitness Goal
- Generates 7-Day Workout Plan

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the server:

python -m uvicorn app:app --reload

Open browser:

http://127.0.0.1:8000
